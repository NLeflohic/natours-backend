## Usage

1. Download the repository

```
$ git clone https://github.com/NLeflohic/natours-backend.git
$ cd natours-backend
```

2. Installation

```
$ yarn install
# or
$ npm install

```

3. Run

```
$ yarn start
# or
$ npm run start
```

Open your favorite web-browser and fill the url : localhost:3000/
